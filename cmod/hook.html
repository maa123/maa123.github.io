フックとは？

何かイベントが起きた時に呼び出される物です。

例えば何かをタップすれば

function useItem(x, y, z, itemid, blockid, side, itemDamage, blockDamage){}
が呼び出されます

  • attackHook

    function attackHook(attacker, victim)

    mobを攻撃した時に発生するイベント

    引数はattacker:攻撃者

    victim:攻撃された側です

  • chatHook

    function chatHook(str)

    プレイヤーがチャットをした時に呼び出されます

    引数はその内容です

  • destroyBlock

    function destroyBlock(x, y, z, side)

    ブロックを破壊した時に呼び出されます

  • eatHook

    function eatHook(hearts, saturationRatio)

    何かを食べた時に呼び出されます

  • entityAddedHook

    function entityAddedHook(entity)

    Entityが追加された時に呼び出されます。

    ワールドに入った時も 追加 という扱いになるのか呼び出されます
