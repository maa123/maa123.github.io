mod-Creater
簡単な物を作ってみましょう
①

最初に完成した物を載せておきます

//ワールドに入るとこんにちは〜と表示
function newLevel(){
clientMessage("こんにちは〜");
}

説明が必要か謎なくらい簡単な物ですが一応

function newLevel(){
}

でワールドに入った時に実行できるようになります。

後は、

clientMessage("こんにちは〜");
で表示するだけです。

②

最初に完成した物を載せておきます

//タップしたら💣爆発
function useItem(x,y,z){
explode(x,y,z,5);
}

5の部分を変えれば爆発力も変わります


function useItem(x,y,z){
}
これは上の物と同じようにタップした時に呼び出される関数です。

この中に書くとタップした時に実行できるようになります。

useItem(x,y,z)のxとyとzは引数という物で、これの場合はどの座標をタップしたかが入っています

explode(x,y,z,5);
これは爆発させる関数で、最初のx,y,zで座標を指定して、5の部分は爆発力が入ります。
