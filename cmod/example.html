mod-Creater
簡単な物を作ってみましょう
①

最初に完成した物を載せておきます

//ワールドに入るとこんにちは〜と表示
function newLevel(){
clientMessage("こんにちは〜");
}

説明が必要か謎なくらい簡単な物ですが一応

function newLevel(){}

でワールドに入った時に実行できるようになります。

後は、

clientMessage("こんにちは〜");
で表示するだけです。

①

最初に完成した物を載せておきます

//タップしたら💣爆発
function useItem(x,y,z){
explode(x,y,z,5);
}

5の部分を変えれば爆発力も変わります
